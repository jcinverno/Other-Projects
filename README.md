# Other-Projects
These are some of the projects I've worked on in various programming languages. Some were for school projects.

<b>QRCode_Reader:</b> This system detects and reads QR codes based on images.It can read an isolated QR code in black and white that is correctly aligned, detect the positioning blocks for code alignment, and locate a QR code on an image that contains additional information.

<b>Magic_Piano:</b> Arduino Projects where keys are associated to different LEDs and musical notes. It creates an electronic keyboard that makes sounds and colors depending on the key pressed.
